# Artificial Intelligence Lab – Programs and Implementations

## Overview

This repository contains implementations of various **Artificial Intelligence (AI) algorithms and problem-solving techniques** using programming languages such as **Python and Prolog**. The aim of this repository is to demonstrate fundamental AI concepts including **search algorithms, game playing strategies, constraint satisfaction problems, and logical reasoning**.

The programs included here are designed as part of **Artificial Intelligence laboratory experiments** to help understand how AI techniques are applied to solve real-world computational problems.

---

## Objectives

* To understand the **core concepts of Artificial Intelligence**.
* To implement classical **AI search algorithms and problem solving techniques**.
* To explore **game playing strategies and optimization techniques**.
* To apply **constraint satisfaction and logical reasoning** in AI problems.
* To analyze and compare the **performance of different algorithms**.

---

## List of Experiments

### Experiment 1

**Magic Square Implementation**

A Python program to generate and verify a **magic square**, where the sum of numbers in each row, column, and diagonal is equal.

---

### Experiment 2

**Water Jug Problem using Production Rules**

This program solves the classic **Water Jug Problem** using production rules to obtain exactly **2 gallons of water** using a **4-gallon jug and a 3-gallon jug**.

---

### Experiment 3

**Tic-Tac-Toe Game Engine with AI Player**

Implementation of a **Tic-Tac-Toe game engine** where a computer AI competes against a human player using decision-making strategies.

---

### Experiment 4

**Graph Search using A* Algorithm and Breadth First Search**

This program demonstrates:

* **A* Algorithm** for finding the shortest path to the goal state.
* **Breadth First Search (BFS)** for exploring graph nodes level by level.
* Comparison of both algorithms based on **time performance and path cost**.

---

### Experiment 5

**Cryptarithmetic Puzzle using Constraint Satisfaction**

Implementation of the famous puzzle:

SEND + MORE = MONEY

using **Constraint Satisfaction Problem (CSP) techniques**.

---

### Experiment 6

**Alpha-Beta Pruning for Game Optimization**

A program that demonstrates **Alpha-Beta Pruning**, an optimization technique used in **Minimax game tree search** to reduce unnecessary computations.

---

### Experiment 7

**Family Relationship using Prolog**

A **Prolog program** that models family relationships such as:

* Parent
* Child
* Sibling
* Grandparent

This demonstrates **logical reasoning and knowledge representation** in AI.
